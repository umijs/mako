Mako `['mɑːkoʊ]` is an **extremely fast**, **production-grade** web bundler based on **Rust**. ✨

## Getting Started

Create a new mako project with the following command.

```bash
$ npm create mako
```

Check out the https://makojs.dev/ for more information.
